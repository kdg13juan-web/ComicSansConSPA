# ChatVengers

SPA de chat con personajes de Marvel (Iron Man, Captain America, Spider-Man) usando Google Gemini AI.

## Stack

- Vanilla JS (ES Modules) + History API
- Express (servidor local)
- Google Gemini API
- Vercel (deploy)

## Local

```bash
npm install
npm run dev
```

## Env

```bash
GEMINI_API_KEY=tu_key
```

## Deploy

Conectar repo a Vercel y agregar `GEMINI_API_KEY` en Environment Variables.
